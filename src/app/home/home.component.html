PÁGINA DE PRUEBA

Esta propuesta consiste en un desarrollo para consulta web de información del reporte entregado en formato excel, por medio de angular y typescript.

Una propuesta se publico en una base de datos MySql y por medio de PHP se consulta la base de datos y se constuye la estructura usada por AMCHARTS.

El segundo modelo esta construido en Microsoft Power BI, utilizando la misma data y herramientas de visualización que incluyen algunas ventajas de interactividad.

Se experimento con gráficas desde el propio Angular pero con cargue estatico de datos únicamente.
